# Whisper CLI

OpenAI Whisper large-v3-turbo를 사용한 자막 생성 도구

## 설치

### 1. 프로그램

- Python 3.10 또는 3.11: https://python.org/downloads (PATH 추가 체크)
- FFmpeg: `choco install ffmpeg` 또는 `scoop install ffmpeg`
- NVIDIA 드라이버: https://nvidia.com/Download/index.aspx (GPU 사용 시)

### 2. CUDA 확인

```bash
nvidia-smi
```

출력에서 CUDA Version 확인 (12.x 이상 권장)
